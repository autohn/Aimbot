# Aimbot
 Aimbot для Overwatch/CSGO,  с помощью части на C++ (DXGI) захватывается экран и происходит наведение, а часть на Python определяет, где находится цель с помощью tensorflow.
 Для работы нужно установить [interception](https://github.com/oblitum/Interception) и положить interception.dll рядом с BT2.exe.
 Работоспособность еще проверить не успел.
